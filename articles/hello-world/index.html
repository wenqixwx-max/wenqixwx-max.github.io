欢迎来到 WenQi 的小宇宙 ✨

如果你看到了这篇文章，说明这个网站已经成功跑起来了。撒花！🎉

为什么会有这个网站？

我曾经用过很多博客平台：知乎、CSDN、Blogger、WordPress……但每一个都让我觉得"差了一点什么"。

  • 有的平台广告太多，读起来分心
  • 有的平台样式统一到没有个性
  • 有的平台说没就没，文章跟着一起消失

所以我决定，自己搭一个。

掌握自己的数据，是数字时代最后的浪漫。

这个网站能做什么？

这是一个完全静态的网站，部署在 GitHub Pages 上：

  1. 3D 角色：首页那只转圈圈的小精灵是用 Three.js 渲染的真实 3D 模型
  2. 后台发布：通过 GitHub API，我可以从任何浏览器发布新文章
  3. 完全开源：所有代码都在我的 GitHub 仓库里，欢迎白嫖

接下来会写什么？

我会在这里记录：

  • 📚 读书笔记
  • 💻 技术折腾
  • 🌱 生活碎片
  • 🎨 设计灵感
  • 🚀 偶尔的胡思乱想

如果你恰好路过这里，希望你能找到一篇让你会心一笑的文章。
